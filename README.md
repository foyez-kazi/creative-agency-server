# Creative Agency - Server

> ### Api for an agency website

## Features

- Users can provide reviews after logged in
- Admin can add new services & new admin
- Users and admins both have different dashboard

## Dependencies

- express
- mongoose
- cors
- dotenv
- express-fileupload
- morgan
